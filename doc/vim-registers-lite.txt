1. vim-registers-lite                  *vim-registers-lite-vim-registers-lite*

The registers.nvim <https://codeberg.org/fosk/registers.nvim> clone for Vim.

- This is a simple porting, not perfect. (I have not seen the source code.)
- The options of registers.nvim are not supported.


REQUIRES                      *vim-registers-lite-vim-registers-lite-requires*

Vim9script


INSTALL                        *vim-registers-lite-vim-registers-lite-install*

>vim
    call dein#add('utubo/vim-registers-lite')
<


USAGE                            *vim-registers-lite-vim-registers-lite-usage*

Type `"` in Normal-mode or `<C-r>` in Insert-mode.

Popup-menu

- `j`, `<C-n>`, `<Tab>`, `<Down>`
    Move the selection down
- `k`, `<C-p>`, `<S-Tab>`, `<Up>`
    Move the selection up
- `g`
    Goto the first line
- `G`
    Goto the last line
- `<CR>`, `<Space>`
    Feedkeys `"{selection}`
- Other keys
    Feedkeys `"{key}`


CONFIGURATION            *vim-registers-lite-vim-registers-lite-configuration*

- `g:registerslite_bind_keys`
    The Dict of key mappings.
    Default:
    `{       'nnoremap': '"',       'noremap!': '<C-r>',       'tnoremap': '<C-w>"',     }`
- `g:registerslite_max_width`
    Maximum width of the window.
    Default: `32`
- `g:registerslite_max_height`
    Maximum height of the window.
- `g:registerslite_delay`
    How long, in seconds, to wait before applying the function.
    Default: `0`
- `g:registerslite_hide_duplicate`
    Hide registers with duplicate value.
    Default: `1`
- `g:registerslite_use_registers`
    Show only specific registers in this order
    Default: `"0123456789abcdefghijklmnopqrstuvwxyz-*~.:%#/`
- `g:registerslite_popup_props`
    You can override popup props.
    See `h: popup_create-arguments`
